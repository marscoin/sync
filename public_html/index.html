Installation Instructions

  1. Stop your Marscoin daemon if it's running: marscoin-cli stop
  2. Backup your wallet.dat file if you have one
  3. Extract the snapshot: tar -xzf marscoin-blockchain-snapshot.tar.gz -C ~/.marscoin/
  4. Start your Marscoin daemon: marscoind
